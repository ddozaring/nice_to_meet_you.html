웹사이트 레이아웃의 종류는 어떤 것이 있을까요?
- 반응형 웹 (rwd,가성비, 유동형 + 적응형)
- 적응형 웹 (awd, 각 화면비 별로 모델링 됨)
- 유동형 웹 (실시간 크기가 변경이 되지만, 레이아웃이 변경 되지는 않음)
- 고정형 웹 (라떼시절)

미디어 쿼리(media query) 구문은 언제 쓸까요?
가로 너비(디바이스 별로 레이아웃 변경시 중요함), 즉 해상도(가로 기준) 구간 구분할 때 사용합니다.
- 해상도 구간
ex) 1구간 1024px : pc 뷰, 2구간 640px : 태블릿 뷰, 320px : 모바일 뷰, 3구간 320px : 모바일뷰
1구간 : 레이아웃 스타일을 데스크탑 스타일로 제작합니다.
2구간 : 레이아웃 스타일을 태블릿 스타일로 제작합니다.
3구간 : 레이아웃 스타일을 모바일 스타일로 제작합니다.
*모바일 경우에는 1px의 밀도가 더 높습니다.*
