01 다음 괄호 안을 채우시오.

( 요소[element/component] )는 HTML 페이지를 구성하는 각 부품입니다.

( 태그 )는 요소를 만들 때 사용하는 작성 방법입니다.

( 주석 )은 프로그램 실행에는 영향을 미치지 않고 설명용으로 사용하는 코드를 의미합니다.

( css[스타일시트] )는 마크업 언어가 표시되는 방법을 기술하는 언어로 HTML 페이지의 스타일을 지정할 때 사용합니다.

( 자바스크립트 )는 HTML 페이지에서 사용하는 프로그래밍 언어를 의미합니다.
